# Simple To Do List Manager

This project is a simple console-based To-Do List Manager written in C.

## Project Description
The program allows the user to manage tasks using a menu-driven interface.

## Features
- Add a task
- View all tasks
- Delete a task
- Exit the program

## Menu
1. Add Task
2. View All Tasks
3. Delete a Task
4. Exit

## Project Structure
- main/main.c : contains the main function
- addTask/addTask.c : contains the addTask function
- viewAlltasks/viewAlltasks.c : contains the viewAlltasks function
- deleteTask/deleteTask.c : contains the deleteTask function
- tasks.h : contains shared declarations and structure
- Simple_To_Do_List_Manager_fullCode.c : contains the full code in one file

## Programming Language
C Programming Language

## Notes
- Tasks are stored using an array of structures.
- When deleting a task, the remaining tasks are shifted correctly.
- Each function is organized in a separate folder as required.
